css 3

It is the latest standard for CSS. Has been split into modules. Contains the old CSS specification & in addition, new modules are added.

Some of the most important CSS3 modules are:

  • Selectors
  • Box Model
  • Backgrounds and Borders
  • Image Values and Replaced Content
  • Text Effects
  • 2D/3D Transformations
  • Animations
  • Multiple Column Layout
  • User Interface

1. borders

(a) border-radius

Used to create rounded corners.

(b) box-shadow

Used to add shadow to boxes.

(c) border-image

Used to put image as a border.
